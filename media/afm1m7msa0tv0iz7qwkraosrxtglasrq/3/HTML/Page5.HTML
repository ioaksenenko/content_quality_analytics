1.1.3 Программы и схемы программ

Рассмотрим отличие программ от схем программ [2].

Схемы программ – это математические модели, описывающие строение программ или, точнее, строение множества программ, где конкретные операции и функции заменены абстрактными функциональными и предикатными символами.

Пример 1.2

Приведем в качестве примера программы вычисления факториала n !  и переворачивания слов. Данный пример поясняет различие между программами и их схемой:

begin
 integer x,y;
 ввод(x);
 y:=1;
L: if x=0 then
 goto L1;
 y:=x*y;
 x:=x-1;
 goto L;
L1: вывод(y);
 end

begin
 string x,y;
 ввод(x);
 y:=ε;
L: if x=0 then
 goto L1;
 y:=CONSCAR(x,y);
 x:=CDR(x);
 goto L;
L1: вывод(y);
 end

begin
 ввод(x);
 y:=a;
L: if p(x) then
 goto L1;
 y:=g(x,y);
 x:=h(x);
 goto L;
L1: вывод(y);
 end

Функция CONSCAR (суперпозиция функций CONS и CAR из языка Лисп) приписывает первую букву первого слова ко второму слову (т. е. CONSCAR(аб,в)=ав), а функция CDR стирает первую букву слова (т. е. CDR(аб)=б).
